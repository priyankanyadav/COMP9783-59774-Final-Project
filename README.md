# 💰 Expense Tracker - Fullstack CRUD Application

## 📌 Project Overview
This is a fullstack Expense Tracker application built using **React (frontend)** and **Node.js / Express (backend)**.  
The project demonstrates full **CRUD (Create, Read, Update, Delete)** functionality with REST API integration.

It allows users to:
- Add expenses
- View all expenses
- Edit existing expenses
- Delete expenses

---

## 🚀 Tech Stack

### Frontend:
- React.js
- JavaScript (ES6+)
- HTML & CSS (inline styling)

### Backend:
- Node.js
- Express.js
- REST API
- UUID for unique IDs
- CORS for cross-origin requests

---

## ⚙️ Features

- ✅ Add new expenses
- ✅ View all expenses
- ✅ Edit existing expenses
- ✅ Delete expenses
- ✅ Real-time UI updates
- ✅ Error handling (try/catch)
- ✅ REST API communication
- ✅ Clean and simple UI

---

## 🔗 API Endpoints

| Method | Endpoint        | Description        |
|--------|----------------|--------------------|
| POST   | /create         | Create expense     |
| GET    | /read           | Get all expenses   |
| GET    | /read/:id       | Get single expense |
| PUT    | /update/:id     | Update expense     |
| DELETE | /delete/:id     | Delete expense     |
